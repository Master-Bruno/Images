Pdf,Png, Jpg,Jpeg,Mp3,Mp4, Opus Etc.. 


 Information Gathering Tool


Installation On Termux
 
 

$ apt update
$ apt upgrade
$ pkg install git
$ pkg install python
$ pkg install perl
$ pip2 install mechanize
$ pip2 install requests
$ git clone https://github.com/achuzachu/Images.git
$ cd lmages
$ ls
$ chmod +x *
$ perl Makefile.PL
$ ./lnformation
